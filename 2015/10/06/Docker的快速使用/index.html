Dockerfile介绍

  1. Dockerfile关键字
    FROM

特别说明
FROM后的镜像特性：
已经存在的镜像
基础镜像
必须是第一条非注释指令

MAINTAINER

指定镜像的作者信息

RUN

指定当前镜像中的运行命令

EXPOSE

指定该镜像容器使用的端口

CMD

在容器中指定运行的命令
RUN是在镜像构建中执行的命令
CMD是在容器运行中执行的命令
CMD是指定容器运行指令的默认行为,当RUN中指定后,CMD指令将会被RUN指令覆盖

ENTRYPOINT

与CMD功能相同,不同的是ENTRYPOINT指令不会被RUN指令覆盖
可以使用docker run –entrypoint强制使用RUN指令覆盖ENTRYPOINT指令

ADD & COPY

将其后的使用文件复制到Dockerfile镜像文件中
ADD vs COPY
ADD包含tar解压功能,如果只是单纯的复制文件,Docker推荐使用COPY

VOLUME [‘/data’]

提供卷的功能,后续讲解

WORKDIR

在容器内部设置执行目录,即CMD,ENTRYPOINT工作目录

ENV

设置环境变量

USER

指定镜像是什么用户运行
