  • 执行结果能理解吗？还是先说下if name == ‘main‘的含义吧
    当包含if name == ‘main‘的程序直接被执行时，如python A.py，此时name与main相等，即if name == ‘main‘成立，执行的是if成立的代码块；
    当包含if name == ‘main‘的程序被import到其他的程序中，如B.py程序中的import A，此时A中的name与main不相等，即if name == ‘main‘不成立，执行的是if不成立的代码块，即else的代码块。
